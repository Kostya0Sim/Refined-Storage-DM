# ![logo](https://github.com/Kostya0Sim/Refined-Storage-DM/blob/main/pack.png) Refined Storage: DM
This resource pack changes the appearance of the [Refined Storage](https://www.curseforge.com/minecraft/mc-mods/refined-storage) GUI to the darker and less saturated tones like in a [Default Dark Mode](https://www.curseforge.com/minecraft/texture-packs/default-dark-mode) resource pack.
